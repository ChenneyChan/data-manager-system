# 水冷设备2 通信协议文档

## 概述

水冷设备2用于外循环冷却系统，通过UDP协议向温升采集程序发送温度和流量数据。
与设备1相比，设备2支持**8个出风温度**检测点。

## 通信参数

| 项目 | 值 |
|------|------|
| 传输协议 | UDP |
| 默认端口 | **8878**（设备1为8877） |
| 目标地址 | 127.0.0.1（本机） |
| 数据编码 | ASCII |
| 字段分隔符 | `;` |
| 发送周期 | 建议 500ms ~ 2000ms |

## 数据格式

```
出水温度;回水温度;流量;环境温度1;环境温度2;出风温度1;出风温度2;出风温度3;出风温度4;出风温度5;出风温度6;出风温度7;出风温度8
```

共 **13个字段**，顺序如下：

| 索引 | 字段 | 单位 | 说明 |
|------|------|------|------|
| 0 | 出水温度 | °C | 冷却水出口温度 |
| 1 | 回水温度 | °C | 冷却水入口温度 |
| 2 | 流量 | L/min | 冷却水流量 |
| 3 | 环境温度1 | °C | 外循环环境温度传感器1 |
| 4 | 环境温度2 | °C | 外循环环境温度传感器2 |
| 5 | 出风温度1 | °C | 外循环出风口温度1 |
| 6 | 出风温度2 | °C | 外循环出风口温度2 |
| 7 | 出风温度3 | °C | 外循环出风口温度3 |
| 8 | 出风温度4 | °C | 外循环出风口温度4 |
| 9 | 出风温度5 | °C | 外循环出风口温度5 |
| 10 | 出风温度6 | °C | 外循环出风口温度6 |
| 11 | 出风温度7 | °C | 外循环出风口温度7 |
| 12 | 出风温度8 | °C | 外循环出风口温度8 |

## 示例报文

```
25.3;22.1;12.5;28.0;27.5;30.1;29.8;30.5;29.2;31.0;28.5;30.0;29.8
```

## 对比设备1

| 对比项 | 设备1 | 设备2 |
|--------|-------|-------|
| UDP端口 | 8877 | 8878 |
| 总字段数 | 9 | 13 |
| 出风温度 | 4个 | 8个 |
| 环境温度 | 2个 | 2个 |
| 流量 | 1个 | 1个 |

## 注意事项

1. **字段精度**：建议保留1位小数（如 `25.3`）
2. **发送频率**：不宜过快（<100ms），建议500ms以上
3. **端口配置**：可在 INI 配置文件的 `[CoolDevice]` 段修改：
   ```ini
   [CoolDevice]
   Port1=8877
   Port2=8878
   SelectedIndex=0
   ```
4. **兼容性**：采集端会自动识别字段数量（9个=设备1，≥9个=设备2）
